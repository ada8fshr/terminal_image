# TerminalImage

TerminalImage is a library to show images on terminals. 

![](./images/example.png)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'terminal_image'
```

## Usage

```ruby
file = File.open('your-image-path.png')
TerminalImage.show(file) # will show image on your terminal
```
